# コンパスウォーク運動・生活記録

片麻痺があり電動車椅子を利用する本人向けの、リハビリデイサービスと生活の個人記録PWAです。左手だけの操作と失語症を考慮し、大きな選択ボタンと短い表示を基本とします。

## 実装済み（第1段階）

- React Router による5画面（ホーム、カレンダー、記録入力、記録詳細、振り返り）のルーティングと共通下部ナビゲーション
- IndexedDB（Dexie）による職員・利用予定・記録・入力下書きのデータモデル
- 初回起動時の職員データ（杉本、松本、八木澤、田村）の登録
- ホーム画面とカレンダー画面
- 月・木を通常利用として一括登録、日付ごとの通常・振替・休み・中止への変更
- スマートフォン優先、主要ボタン54px以上、PWAマニフェスト

## データモデル

| テーブル | 内容 |
| --- | --- |
| `staff` | 氏名、職種、登録日時 |
| `schedules` | 日付、利用種別、メモ、更新日時 |
| `records` | 天気、体調、バイタル、運動、担当者、生活変化、メモ |
| `drafts` | 入力途中の記録 |

## 未実装
